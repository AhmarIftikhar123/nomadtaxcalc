Why Each Page Matters
🔴 /about — AdSense Hard Requirement
Google AdSense reviewers manually check for an About page. It must explain:

Who runs the site
What the site does
Why users should trust it

Without this, AdSense auto-rejects.
🔴 /privacy-policy — AdSense Hard Requirement
AdSense requires a privacy policy because it uses cookies and collects data. Must include:

What data you collect
How you use cookies
Third-party advertising disclosure (AdSense itself)
User rights (GDPR if EU users)

🔴 /contact — Trust Signal
Both AdSense and Google Search Quality raters look for this. Shows the site is real and owned by a real person.
🔴 / (Landing Page) — SEO Foundation
This is your money page. Google ranks this for keywords like "digital nomad tax calculator". Needs the strongest SEO treatment.
🟡 /tax-calculator — Tool Page
This is what users actually use. Needs schema markup to appear as a rich result in Google (WebApplication schema).
🔴 /tax-calculator/shared/{token} — Must be NOINDEX
Each token generates a unique URL. If Google crawls thousands of these, it will treat your site as duplicate/thin content and penalize rankings.

STEP 1 — Create SEO Service Class (The Brain)
Purpose: Central place to define all meta tags per route. Keeps controllers clean.
php<?php
// app/Services/SeoService.php

namespace App\Services;

class SeoService
{
    protected array $data = [];

    public function __construct()
    {
        // Safe defaults for every page
        $this->data = [
            'title'            => 'Nomad Tax Calculator — Free Tax Tool for Digital Nomads',
            'description'      => 'Calculate your taxes as a digital nomad in seconds. Compare tax rates across 50+ countries. Free, instant, no signup required.',
            'canonical'        => url()->current(),
            'og_image'         => asset('images/og-default.jpg'),
            'og_type'          => 'website',
            'robots'           => 'index, follow',
            'schema'           => null,
        ];
    }

    public function set(array $data): self
    {
        $this->data = array_merge($this->data, $data);
        return $this;
    }

    public function noIndex(): self
    {
        $this->data['robots'] = 'noindex, nofollow';
        return $this;
    }

    public function get(): array
    {
        return $this->data;
    }
}
Why: Every controller calls this service instead of manually writing meta arrays. One place to change everything.

STEP 4 — Generate Sitemap
Purpose: Tell Google exactly which pages exist and how important they are. Without this, Google discovers pages slowly or misses them.
php<?php
// app/Console/Commands/GenerateSitemap.php

namespace App\Console\Commands;

use Illuminate\Console\Command;
use Spatie\Sitemap\Sitemap;
use Spatie\Sitemap\Tags\Url;

class GenerateSitemap extends Command
{
    protected $signature   = 'sitemap:generate';
    protected $description = 'Generate the XML sitemap';

    public function handle(): void
    {
        Sitemap::create()
            // Highest priority — your money pages
            ->add(Url::create('/')
                ->setPriority(1.0)
                ->setChangeFrequency('weekly'))
            ->add(Url::create('/tax-calculator')
                ->setPriority(0.95)
                ->setChangeFrequency('monthly'))
            ->add(Url::create('/tax-calculator/compare')
                ->setPriority(0.90)
                ->setChangeFrequency('monthly'))

            // Trust pages — important for AdSense
            ->add(Url::create('/about')
                ->setPriority(0.7)
                ->setChangeFrequency('yearly'))
            ->add(Url::create('/contact')
                ->setPriority(0.6)
                ->setChangeFrequency('yearly'))
            ->add(Url::create('/privacy-policy')
                ->setPriority(0.5)
                ->setChangeFrequency('yearly'))

            // NOTE: Do NOT add /tax-calculator/shared/* or auth routes
            ->writeToFile(public_path('sitemap.xml'));

        $this->info('Sitemap generated successfully.');
    }
}
Run once, then add to shared hosting cron weekly:

```
### STEP 5 — robots.txt

**Purpose:** Tells crawlers what to index and what to ignore. Prevents Google wasting crawl budget on auth pages and API endpoints.
```
# public/robots.txt

User-agent: *
Allow: /

Allow: /tax-calculator/compare
Allow: /about
Allow: /contact

# Block everything that should not be indexed

Disallow: /profile
Disallow: /my-calculations
Disallow: /tax-calculator/shared/

Sitemap: https://nomadtaxcalc.com/sitemap.xml

```
### STEP 6 — AdSense-Required Pages Content Guide

**Purpose:** AdSense reviewers read these pages manually. Thin or missing content = instant rejection.

**`/about` must contain:**
- Who you are (name or team name)
- Why you built this tool
- Who it helps (digital nomads, freelancers, remote workers)
- That tax info is for guidance only (legal disclaimer)

**`/privacy-policy` must contain:**
- What data is collected (email if newsletter, usage analytics)
- That Google AdSense/Analytics uses cookies
- How to opt out
- GDPR mention if you have EU users
- Contact email for data requests

**`/contact` must contain:**
- A working contact form (you already have POST `/contact`)
- Response time expectation
- Alternatively, a real email address
```

2. Add property → Domain → nomadtaxcalc.com

4. Go to Sitemaps → Submit: https://nomadtaxcalc.com/sitemap.xml
